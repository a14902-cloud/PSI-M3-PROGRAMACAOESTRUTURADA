# PSI-M3-PROGRAMA-AOESTRUTURADA
# Exercícios de Python – Módulo 3

Este repositório contém os exercícios desenvolvidos em **Python**, no âmbito do **Módulo 3** da disciplina **Programação de Sistemas Informáticos**, do curso profissional **Técnico de Gestão e Programação de Sistemas Informáticos (TGPSI)**.

Os exercícios estão organizados por **atividades (A01 a A04)**, facilitando a consulta e a avaliação da progressão dos conteúdos.

---

## 📂 Estrutura do Repositório
---

## 📁 A01 – Fundamentos e Estruturas de Decisão

### Conteúdos:
- Função `print()`
- Variáveis e tipos de dados
- Entrada de dados com `input()`
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `elif`, `else`)

### Exercícios:
- Impressão de mensagens
- Cálculo de idade
- Soma e operações matemáticas
- Conversão de temperatura
- Verificação de par/ímpar
- Comparação de números
- Classificação etária
- Sistema simples de login
- Cálculo de descontos, IMC e ano bissexto

---

## 📁 A02 – Ciclos de Repetição

### Conteúdos:
- Ciclo `while`
- Ciclo `for`
- Contadores
- Intervalos numéricos
- Repetições controladas por condição

### Exercícios:
- Números de 1 a 10
- Contagem decrescente
- Números pares
- Tabuada
- Soma de números
- Menu simples com opções

---

## 📁 A03 – Listas e Programas Interativos

### Conteúdos:
- Listas (`list`)
- Inserção e remoção de dados
- Percorrer listas
- Menus interativos
- Cálculos com conjuntos de dados

### Exercícios:
- Gestão de notas
- Listagem de valores
- Cálculo da média
- Filtragem de notas positivas
- Cálculo de fatorial

---

## 📁 A04 – Projeto Final: Gestão de Stocks

### Conteúdos:
- Funções em Python
- Dicionários
- Programação modular
- Menus estruturados
- Validação de dados

### Funcionalidades:
- Adicionar materiais ao stock
- Consultar stock de um material
- Atualizar quantidades (entrada/saída)
- Exibir estado geral do stock
- Menu interativo em consola
